# PPT 自动生成 Web 系统 - 性能与缓存策略

## 1. 概述

### 1.1 性能目标

| 指标 | 目标值 |
|------|--------|
| 页面首屏加载 | < 2 秒 |
| 缩略图列表滚动 | 60fps |
| SVG 主预览切换 | < 200ms |
| 10 页 PPT 完整生成 | < 5 分钟 |
| 单页聊天编辑响应 | < 30 秒 |
| PPT 导出（10 页） | < 15 秒 |

### 1.2 性能瓶颈分析

| 环节 | 瓶颈 | 影响 |
|------|------|------|
| LLM 调用 | 单次 2-15 秒，多 Agent 累计 | 生成总时长 |
| SVG 渲染 | 复杂 SVG DOM 计算量大 | 预览流畅度 |
| 图片搜索+下载 | 外部网络延迟 | 视觉处理阶段时长 |
| 导出渲染 | SVG → Canvas → PNG 逐页转换 | 导出等待时间 |
